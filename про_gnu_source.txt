1) _GNU_SOURCE:
- _DEFAULT_SOURCE (_BSD_SOURCE + _SVID_SOURCE в старых версиях)
- _POSIX_C_SOURCE=200809L
- _XOPEN_SOURCE=700
- GNU-специфичные расширения

2) 1990-е: 
#define _GNU_SOURCE
BSD + System V + GNU

Чичас:
#define _GNU_SOURCE  
POSIX + X/Open + BSD + System V + GNU

3) #define _DEFAULT_SOURCE  // _BSD_SOURCE + _SVID_SOURCE
